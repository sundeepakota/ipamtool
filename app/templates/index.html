How to Use:

• You can perform searches using CIDR notation (e.g., 192.168.1.0/24) to search for specific locations or comments.

• To search by location or comments, enter relevant keywords (e.g., Office, Data Center) in the search box.

• Note: For CIDR notation, use the format IP_ADDRESS/NETMASK_LENGTH (e.g., 192.168.1.0/24).
